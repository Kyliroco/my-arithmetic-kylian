PGCD Module¶

Le module PGCD contient des fonctions pour calculer le plus grand commun diviseur.

my_arithmetic_kylian.pgcd.pgcd(a, b)¶

Return the greatest common divisor of a and b

Paramètres:

  • a (int) – First number

  • b (int) – Second number

Renvoie:

The greatest common divisor of a and b

Type renvoyé:

int

Exemples

>>> pgcd(10, 25)
5
